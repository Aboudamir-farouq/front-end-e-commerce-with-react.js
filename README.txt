ce repo représente le contenu du dossier "src" 
créé à l'aide de la commande : npx create-react-app my-app

aprés la création du create-react-app modifier le contenu 
du "src" par ces fichiers

les bibliothéques utilisées son "react-bootstrap" et "axios"
ce qui veut dire qu'il faut éxécuter les commandes suivantes:
==> npm install react-bootstrap bootstrap
==> npm install axios

puisque on a travaillé avec bootstrap il faut également 
insérer la balise suivante au sein de la balise <head> 
du fichier index.html qui se trouve dans le dossier public
<link rel="stylesheet" href="https://cdn.jsdelivr.net/npm/bootstrap@5.2.2/dist/css/bootstrap.min.css" integrity="sha384-Zenh87qX5JnK2Jl0vWa8Ck2rdkQ2Bzep5IDxbcnCeuOxjzrPF/et3URy9Bv1WTRi" crossorigin="anonymous" />
